# 遊戲操作說明 (Game Controls)

## 基本操作

### 移動
- **左鍵點擊**：移動玩家到指定位置
- **移動時轉向**：移動時角色會自動轉向移動目標

### 自動射擊
- **自動射擊**：當範圍內有敵人時，角色會自動射擊
- **自動瞄準**：射擊時會自動轉向最近的敵人
- **射擊頻率**：受武器射擊冷卻時間限制（基礎 0.5 秒）

### 放置炮塔
1. 按 **T 鍵** 進入炮塔放置模式
2. 在放置模式下，**左鍵點擊** 地圖上的位置來放置炮塔
3. 按 **ESC 鍵** 或再次按 **T 鍵** 取消放置模式
4. 炮塔只能放置在距離基地一定距離的位置（至少 8 單位）
5. 炮塔之間需要保持最小間距（3 單位）

## UI 按鈕操作

### 升級武器
- 點擊底部控制面板的 **"升級武器 (5)"** 按鈕
- 需要消耗 5 個資源
- 升級後武器傷害和射程都會增加

### 升級炮塔
- 點擊底部控制面板的 **"升級炮塔 (10)"** 按鈕
- 需要消耗 10 個資源
- 會自動升級你擁有的**等級最低**的炮塔（如果有多個炮塔，會優先升級等級最低的）
- 升級後炮塔傷害和射程都會增加

### 放置炮塔（UI 方式）
- 點擊底部控制面板的 **"放置炮塔 (T)"** 按鈕
- 這會切換炮塔放置模式（等同於按 T 鍵）
- 放置模式下按鈕會變成綠色並顯示 "取消放置"

## 遊戲機制

### 資源獲得
- 擊敗怪物可以獲得資源
- 每個怪物基礎獎勵為 1 資源
- 炮塔擊敗的怪物也會給炮塔擁有者資源

### 武器系統
- 基礎傷害：5
- 基礎射程：20 單位
- 射擊冷卻：0.5 秒
- 每級升級增加 2 傷害和 2 射程

### 炮塔系統
- 基礎傷害：3
- 基礎射程：15 單位
- 射擊冷卻：1 秒
- 每級升級增加 1 傷害和 1 射程
- 炮塔會自動瞄準並攻擊範圍內最近的怪物

### 基地系統
- 基地位於地圖中心（64, 36）
- 基地基礎生命值：100
- 怪物到達基地會造成 1 點傷害
- 基地生命值歸零遊戲結束

### 怪物系統
- 怪物每 5 秒生成一次
- 怪物從地圖邊緣隨機位置生成
- 怪物會自動朝基地中心移動
- 怪物基礎生命值：10
- 怪物基礎移動速度：0.5 單位/秒

## 快捷鍵總結

| 按鍵 | 功能 |
|------|------|
| 左鍵 | 移動 |
| T | 切換炮塔放置模式 |
| ESC | 取消炮塔放置模式 |

**注意**：射擊現在是自動的，當範圍內有敵人時會自動射擊和瞄準。

## 提示

1. **資源管理**：優先升級武器，因為玩家可以主動攻擊，而炮塔是被動防禦
2. **炮塔位置**：將炮塔放置在怪物路徑上可以最大化傷害輸出
3. **射擊技巧**：射擊時會自動瞄準範圍內最近的怪物，不需要精確瞄準
4. **基地防禦**：保護基地是首要任務，如果基地生命值歸零，遊戲結束
