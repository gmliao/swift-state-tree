feat: implement transport protocol optimizations (Phase 1-3)

This commit implements core infrastructure for transport protocol optimization,
including schema hash verification, path hash generation, and player slot allocation.

## Phase 1: Schema Hash Verification

- Add DeterministicHash utility with FNV-1a algorithm
- Generate schemaHash in ProtocolSchema during codegen
- Validate client schemaHash on join (prevent version mismatches)
- Add joinSchemaHashMismatch error code
- Update TypeScript SDK to send schemaHash in join metadata

## Phase 2: Path Hash Generation (Schema Only)

- Implement PathFlattener to traverse JSON Schema tree
- Generate path patterns with wildcards (e.g., "players.*.position.x")
- Add pathHashes field to LandSchema (54 paths in GameDemo)
- Create PathHasher utility for path splitting
- Update TypeScript schema types to include pathHashes
- Note: Encoder/Decoder integration deferred to next commit

## Phase 3: PlayerSlot Allocation

- Add deterministic slot allocation using FNV-1a hash
- Implement bidirectional mapping (slot ↔ playerID)
- Include playerSlot in JoinResponse payload
- Add slot cleanup on disconnect
- Update TypeScript transport types
